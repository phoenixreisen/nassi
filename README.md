# nassi

An application to generate beautiful nassi-shneiderman diagrams from textual
representations.


## Usage

The executable JAR is in the `bin` directory.

```
$ java -jar bin/nassi.jar
```


## Build standalone JAR file

```
$ lein clean; lein uberjar
```
